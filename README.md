# Travel Companion App

##  Overview
The Travel Companion App is an Android application built using **Kotlin** and **Jetpack Compose**.  
It helps travellers perform quick and simple unit conversions for everyday travel needs.

The app supports:
- Currency conversion
- Fuel / Distance / Liquid conversion
- Temperature conversion

---

## Features
- Modern UI using Jetpack Compose
- Category-based conversion system
- Dropdown selection for units
- User input field
- Instant conversion results
- Input validation and error handling
- Dynamic UI color based on category
- Reusable UI components

---

##  Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Material 3 UI

---

##  Conversion Categories

### 1. Currency
Supported units:
- USD, AUD, EUR, JPY, GBP

### 2. Fuel / Distance / Liquid
Supported conversions:
- mpg ↔ km/L
- Gallon (US) ↔ Liters
- Nautical Mile ↔ Kilometers

### 3. Temperature
Supported units:
- Celsius, Fahrenheit, Kelvin

---

##  Validation Features
The app includes:
- Empty input validation
- Numeric input validation
- Same unit conversion handling
- Negative value restriction (for currency & fuel)
- Error messages for unsupported conversions

---

##  How It Works
1. Select a category  
2. Choose source unit  
3. Choose destination unit  
4. Enter a value  
5. Click **Start Conversion**  
6. View the result  

---

##  How to Run the Project
1. Clone the repository

2. Open in Android Studio

3. Sync Gradle

4. Run on emulator or physical device
